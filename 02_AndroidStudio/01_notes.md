# ⚙️ Android Studio Setup

This section covers the installation and setup of tools required for Android Development.

---

## 📌 Prerequisites

### 1. Install JDK (Java Development Kit)
- Required to write and run Java programs
- Android apps (in this repo) are developed using Java

---

## 2. 🛠️ Install Android Studio

Download and install:
👉 https://developer.android.com/studio

### 3. What is Android Studio?

 Android Studio is the official Integrated Development Environment `(IDE)` for building Android applications, developed by : `Google`

## 🚀 Why Use Android Studio?

- Provides a powerful code editor for Java/Kotlin  
- Supports XML-based UI design  
- Includes Emulator (virtual Android device)  
- Built-in debugging and testing tools  
- Easy project management  
---
## 📂 Basic Project Structure

When a project is created, it contains:

### 🔹 MainActivity.java
- Entry point of the app  
- Contains application logic  

### 🔹 activity_main.xml
- Defines the UI (User Interface)  
- Contains elements like TextView, Button  

### 🔹 AndroidManifest.xml
- Configuration file  
- Declares app components and permissions  

### 🔹 res/ folder
- Stores resources like layouts, images, strings  

### 🔹 Gradle Scripts
- Used for building and managing dependencies  

---
## 🎯 Summary

- JDK is required for Java development  
- Android Studio is the main tool for building Android apps  
- Android apps are built using Java (logic) + XML (UI)  
- Understanding project structure is important for development
