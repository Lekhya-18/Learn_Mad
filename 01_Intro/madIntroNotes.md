# Introduction to Mobile App Development

## Mobile Application
👉  App which is Built to run on mobile devices (Android / iOS).
- Mainly we use 3 Operating Systems : 
    - Android
    - iOS (iphone OS)
    - Windows 
## What is MAD?
Mobile App Development is the process of creating apps for mobile devices.

## Types of Apps

### 1. Native App
- Built using Java/Kotlin for Android
- High performance

### 2. PWA
- Web-based apps
- Works in browser

### 3. Cross-Platform
- One code for multiple platforms
